Xmas Mode

This project consists on an automatized Christmas tree (using an ESP8266), which you can turn on and off from your phone. Powered by a homemade MQTT network, using an Orange Pi as a "broker" (server).


Materials:

WiFi IoT Relay Board based on ESP8266 Orange Pi One

Software:

Arduino IDE (Program ESP8266) Android Studio (Android app) Eclipse Mosquitto (MQTT Server) Eclipse Paho (MQTT Client) PubSubClient Library (ESP8266 MQTT Client) Anjuta IDE (GTK+ Linux application)
